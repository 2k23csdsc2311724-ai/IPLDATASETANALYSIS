Project Overview
----------------
Analyze IPL match data to find patterns in team performance, stadium scoring trends, and toss impact.

Tools Used
----------
Python
Pandas
Matplotlib
Seaborn

Visualizations
--------------
• Matches per season
• Highest scoring stadiums
• Team wins distribution
• Toss decision analysis
• Player performance

Key Insights
------------
• Mumbai Indians have the highest wins.
• Some stadiums host high scoring matches.
• Toss does not strongly determine match outcomes.
